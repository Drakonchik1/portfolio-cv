# Pavlo Portfolio CV

This folder contains your standalone portfolio website and web CV.

## Run locally

```bash
npm install
npm run dev
```

Dev server runs on: `http://localhost:5174`

## Build for production

```bash
npm run build
```

## Main files

- `src/App.jsx` - portfolio content and sections
- `src/App.css` - visuals, cards, animation effects
- `CV_Pavlo_Dorofieiev_ENG.md` - editable CV text
